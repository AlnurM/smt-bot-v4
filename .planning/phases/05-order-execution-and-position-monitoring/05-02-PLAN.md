    **`monitor_positions(session_factory, binance_client, settings, bot)`:**
    - Load all open, non-dry-run positions that have sl_order_id and tp_order_id not null.
    - Process each position sequentially (not as concurrent tasks — prevents win streak race condition per Pitfall 8).
    - Wrap each position in try/except to isolate failures.
    - On BinanceAPIException with code -2013: mark position.status = 'orphaned', log warning, no Telegram alert.
    - On other exceptions: log error, continue to next position.

    **`_handle_position_close(binance_client, session_factory, bot, settings, position, close_reason, filled_order_id, surviving_order_id, exit_price)`:**

    Sequential steps:
    1. Cancel surviving bracket order: `futures_cancel_order(symbol=position.symbol, orderId=int(surviving_order_id))`. Catch BinanceAPIException code -2011 (already cancelled/expired) and continue. Log warning for other codes.
    2. Fetch realized PnL: `futures_account_trades(symbol=position.symbol, limit=10)`. Sum `float(t["realizedPnl"])` for trades where `t["orderId"] == int(filled_order_id)`. This is the definitive PnL including fees.
    3. DB writes (all in one session):
       a. Create Trade row: symbol, side=position.side, entry_price=position.entry_price, exit_price=exit_price, realized_pnl=realized_pnl, close_reason=close_reason, opened_at=position.created_at, closed_at=datetime.now(timezone.utc), position_id=position.id.
       b. Update position.status = 'closed', position.updated_at = now.
       c. Upsert DailyStats using pg_insert ON CONFLICT DO UPDATE (exact pattern from interfaces block above). After upsert, re-query and compute win_rate.
       d. Load RiskSettings (single row). If close_reason == 'tp': risk.win_streak_current += 1; risk.current_stake_pct = get_next_stake(risk.win_streak_current, risk.progressive_stakes, risk.base_stake_pct, risk.wins_to_increase). If close_reason == 'sl': risk.win_streak_current = 0; risk.current_stake_pct = get_stake_after_loss(risk.base_stake_pct). risk.updated_at = now. Commit all.
    4. Send Telegram close notification (template from interfaces block above).

    **`_update_unrealized_pnl(binance_client, session_factory, position)`:**
    - `futures_position_information(symbol=position.symbol)` → list. Find entry where pi["symbol"] == position.symbol.
    - If positionAmt == 0: position may have closed without bot detecting it — log warning, do not change status (monitor will catch it on next cycle via order status check).
    - Otherwise: update position.unrealized_pnl = float(pi["unRealizedProfit"]), position.current_price = float(pi["markPrice"]), position.updated_at = now. Commit.
