    ```python
    async def handle_confirm(callback: CallbackQuery, callback_data: SignalAction, **kwargs):
    ```
    1. `await callback.answer()` — FIRST — answers within Telegram's 60s deadline (Pitfall 1)
    2. Open session from `kwargs["session_factory"]`

       ```python
       result = await session.execute(
           select(Signal).where(
               Signal.id == uuid.UUID(callback_data.signal_id),
               Signal.status == "pending",
           ).with_for_update()
       )
       signal = result.scalar_one_or_none()
       ```
    4. If signal is None (already confirmed/rejected/expired — double-tap):
       - Try `await callback.message.edit_reply_markup(reply_markup=None)` wrapped in try/except
